A C library inspired by _C Interfaces and implementations_.

Current support:

- memory management;
- hash function;
- closure;
- list;
- atom;
- hash table;
- assoc table(linear table);
- resizable array;
- deque;
- resizable string;
- state machine(a subset of [Harel 1987]).

Following development is moved to [litar](www.github.com/chu-mirror/litar)
as its standard library.
